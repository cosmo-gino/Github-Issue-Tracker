<h3>1️⃣ What is the difference between var, let, and const</h3>
<p>var is function-scoped and can be re-declared and updated, while let and const are block-scoped. let can be updated but not re-declared, and const cannot be updated or re-declared.</p>

<h3>2️⃣ What is the spread operator (...)?</h3>

<p>The spread operator (...) is used to expand the elements of an array or object. It is commonly used to copy or merge arrays and objects.</p>

<h3>3️⃣ What is the difference between map(), filter(), and forEach()?</h3>

<ul><il>map() creates a new array by transforming each element.</il> 
 <il>filter() creates a new array with elements that pass a condition.</il> 
<il>forEach() just loops through the array and does not return a new array.</il>
 </ul>

 <h3>4️⃣ What is an arrow function?</h3>
 <p>An arrow function is a short way to write a function in JavaScript. It uses the => symbol and makes the code shorter.</p>

  <h3>5️⃣ What are template literals?</h3>

<p>Template literals are a way to write strings in JavaScript using backticks (` `). we can easily put variables inside using ${}.</p>
